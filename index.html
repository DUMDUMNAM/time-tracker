Google Sheets 연동
연동 방법
  1. 스프레드시트 URL — Google Sheets를 열고 주소창의 URL 전체를 복사해서 붙여넣으세요.
  2. Apps Script 웹앱 URL 가져오는 방법:
    1. Google Sheets에서 확장 프로그램 → Apps Script 클릭
    2. Apps Script 편집기에서 배포 → 배포 관리 클릭
    3. 웹 앱 URL을 복사해서 붙여넣으세요
    4. 아직 배포가 없다면: 배포 → 새 배포 → 유형을 웹 앱으로 선택 → 액세스 권한을 모든 사용자로 설정 → 배포 클릭 후 URL 복사
